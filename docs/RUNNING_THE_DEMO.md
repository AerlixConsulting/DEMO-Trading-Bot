# Running the Demo

## Requirements

- Python 3.10 or newer
- Git
- Terminal access

## Setup

```bash
git clone https://github.com/AerlixConsulting/DEMO-Trading-Bot.git
cd DEMO-Trading-Bot
python -m venv .venv
source .venv/bin/activate
cp .env.example .env
python src/demo_bot.py
```

On Windows:

```bash
.venv\\Scripts\\activate
```

## Notes

This demo uses mock data and does not require credentials.
